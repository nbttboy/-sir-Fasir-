# 发票sir - 您的私人 AI 发票警察 👮‍♂️

**发票sir (Fasir)** 是一款专为财务、出纳及商务人士打造的、基于 AI 视觉大模型的智能财税风控应用。它不仅是一个 OCR 工具，更是深度集成财税逻辑、法律合规与 AI 代理（Agent）的数字化风控终端。

## 🌟 核心杀手锏

### 1. 🤖 双 Agent 协同审核
*   **国税局查验 Agent**: 模拟官方查验流程，不仅核对发票票面，更深度验证发票在国税系统的实时状态，识别“克隆票”与“废票”。
*   **虚开风险专项 Agent**: 检索工商登记信息及公开信用数据，通过 AI 逻辑分析开票方是否具备实际经营能力。识别“空壳公司”、“地址异常”、“经营范围不符”等虚开红线风险。**坚持真实原则，拒绝虚构，无法获取时如实提示。**

### 2. 🛡️ 隐私零泄露：本地大模型支持
*   支持连接本地部署的 **Ollama** 或 **LM Studio**。
*   财务数据不离终端，在本地完成视觉解析与风险审计，彻底消除企业敏感数据外泄风险。

### 3. 📊 批量风控与合规审计
*   **连号预警**: 智能判定多张发票是否属于同一业务拆单，规避税务合规风险。
*   **企业制度解析**: 语义化导入企业报销制度，由 AI 自动判定该笔报销是否符合公司行政规定。
*   **发票类型全覆盖**: 支持增值税专票/普票、卷票、出租车、火车票、定额发票等多种票种。

## 🚀 快速开始

### 本地开发运行
1. **克隆仓库**
   ```bash
   git clone https://github.com/nbttboy/-sir-Fasir-
   cd sir-Fasir
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置环境变量**
   在项目根目录创建 `.env` 文件，并配置您的 API Key（如果使用 Google Gemini）：
   ```env
   VITE_API_KEY=您的_GEMINI_API_KEY
   ```
或者直接在系统配置界面填写

4. **启动开发服务器**
   ```bash
   npm run dev
   ```
   启动后，在浏览器访问 `http://localhost:3000`。



### 移动端集成 (PWA)
应用完全支持 PWA。在移动浏览器中打开后，点击“添加到主屏幕”，即可像原生 App 一样享受流畅的拍照报销体验。

## 🛠️ 本地模型配置 (推荐)
为了极致的隐私保护，建议使用具备 Vision 能力的模型：
1. **Ollama**: `ollama run llama3.2-vision`
2. **配置**: 在本应用“设置”中填写 Base URL 为 `http://localhost:11434/v1`。
